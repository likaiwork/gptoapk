## 小米 / 三星：如何开启「未知来源」（Android 12+）

无论哪个国家，从浏览器下载 APK 后都需要给**当前使用的 App**（Chrome、Files 等）安装权限。

### 三星 One UI

1. 设置 → 应用程序 → 右上角 ⋮ → **特殊访问权限**
2. **安装未知应用** → 选择你用来下载/打开 APK 的应用（如 Samsung Internet、Chrome）
3. 打开 **允许来自此来源的应用**
4. 回到「我的文件」或下载通知，点击 APK 安装

### 小米 MIUI / HyperOS

1. 设置 → 隐私保护 → **特殊权限设置** → **安装未知应用**
2. 选择浏览器或「文件管理」→ 允许
3. 若安装失败：设置 → 更多设置 → 开发者选项 → 关闭 **MIUI 优化**（装完可再打开）
4. 部分机型需在「安全」里关闭对未知应用的拦截提示
